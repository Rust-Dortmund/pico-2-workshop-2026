# APDS-9960

You can find the datasheet for the APDS-9960 [here](https://cdn.sparkfun.com/assets/learn_tutorials/3/2/1/Avago-APDS-9960-datasheet.pdf) (unfortunately, we can't directly reproduce it here).

In addition, the manufacturer of the breakout board SOLDERED provides online documentation on the board we use on [this page](https://oldshop.soldered.com/documentation/apds-9960/overview/).
Note that there are multiple pages of documentation, including a very useful "Hardware details" page on which you will also find a link to [their GitHub repository](https://github.com/SolderedElectronics/Color---gesture-sensor-APDS-9960-breakout-hardware-design) containing the [schematics](https://github.com/SolderedElectronics/Color---gesture-sensor-APDS-9960-breakout-hardware-design/blob/main/OUTPUTS/V1.1.1/Color%20and%20gesture%20sensor%20APDS-9960%20breakout%20Schematics.pdf) for their wiring for the electrically inclined.
